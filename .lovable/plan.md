

# PISTACHIO – Coffee Shop & Crêpe Menu Website

## Overview
A modern, mobile-first one-page menu website for PISTACHIO Coffee Shop & Crêpe, optimized for QR-code scanning and mobile browsing. No backend needed — fully static.

---

## 1. Header & Navigation
- **Sticky top navigation bar** with the PISTACHIO logo (from uploaded image) on the left
- Navigation links: **Menu**, **Contact**, **Location** — smooth-scroll to sections
- Mobile-friendly with a hamburger menu or compact nav
- Primary brand color `#9fbd31` used for accents

## 2. Hero Section
- Eye-catching hero area with the PISTACHIO logo prominently displayed
- Tagline/welcome text with café-style typography
- Clean, appetizing aesthetic with the brand's green, black, and white palette

## 3. Menu Section (Core Feature)
- **10 collapsible accordion categories**: Hot Drinks, Frappe, Milkshakes, Smoothies, Iced Drinks, Iced Tea, Refreshers, Crêpes, Cocktails, Croissants
- Each category expands to show items with:
  - Item name
  - Short appetizing description
  - Price ($4–$15 range)
- Smooth accordion open/close animations
- Touch-friendly, large tap targets for mobile
